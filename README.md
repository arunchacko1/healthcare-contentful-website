# Everwell Family Clinic

A fictional healthcare clinic marketing website built with Next.js, React, TypeScript, Tailwind CSS, and Contentful.

The project is built in small milestones. Each milestone adds a focused layer of functionality and includes documentation, review notes, and quiz questions.

## Current Milestone

Milestone 1: Project foundation.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
```

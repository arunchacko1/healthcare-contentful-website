# Everwell Family Clinic

A fictional healthcare clinic marketing website built with Next.js, React, TypeScript, Tailwind CSS, and Contentful.

The project is built in small milestones. Each milestone adds a focused layer of functionality.

## Current Milestone

Milestone 9: SEO, schema, accessibility, and performance pass.

See `docs/milestone-9-seo-schema-a11y-performance.md` for implementation
notes, review guidance, and quiz questions.
